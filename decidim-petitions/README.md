# Decidim::Petitions

Add the functionality to work with DECODE petitions.

## Usage

Petitions will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'decidim-petitions
```

And then execute:

```bash
bundle
```
